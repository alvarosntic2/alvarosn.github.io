INTRODUCCIÓN A CSS

Álvaro Sánchez

Una vez que tenemos mostrado el contenido deseado en nuestra página web, el siguiente paso es estilizarlo. Para ello, necesitaremos cambiar el lenguaje HTML por el de CSS.
Lo podemos utilizar de distintas maneras:

Cambiando el estilo de los elementos de un tipo etiqueta

Por ejemplo: Todos los párrafos de la página tendrán el azul marino como color de fuente y la página tendrá un fondo naranja apagado.

Cambiando el estilo de un elemento concreto

En este caso, voy a hacer que solo el encabezado de este apartado cambie su color de fuente a un tono verdoso.

Cambiando el estilo de varios elementos concretos

Ahora, voy a hacer que solo el fondo de los párrafos de los encabezados tengan un distinto color.
